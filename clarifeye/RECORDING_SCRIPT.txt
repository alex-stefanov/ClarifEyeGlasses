ClarifEye Audio Recording Script
=================================

Recording instructions:
  - Use a quiet environment, decent USB mic
  - Record at 22050 Hz or higher, mono preferred
    (we will downsample if needed)
  - Save as WAV (uncompressed) -- file format conversion will be
    done by the tools/process_recordings.py script
  - Trim silence at start/end during your editing pass, or let the
    processing script do it (recommended: let the script handle it)
  - One file per phrase, named exactly as shown
  - Speak naturally, not robotically.
    Imagine you are guiding a friend.

Save Bulgarian recordings to: data/audio/bg/<filename>.wav
Save English recordings to:   data/audio/en/<filename>.wav

Then run: python tools/process_recordings.py
